# 🎮 GameCore - Tienda de Videojuegos (Frontend)

Bienvenido a **GameCore** 👋
Este proyecto es una tienda de videojuegos desarrollada con fines visuales, enfocada en el diseño moderno, la experiencia de usuario y la presentación de productos.

---

## 🛍️ Sobre el proyecto

GameCore es una página web que simula una tienda online de videojuegos.
Está diseñada únicamente a nivel **frontend**, por lo que no incluye funcionalidades reales como pagos, carrito o base de datos.

El objetivo principal es mostrar la estructura y el diseño de una tienda digital.

---

## 🎨 Características

* Diseño moderno y atractivo
* Catálogo visual de videojuegos
* Secciones organizadas (productos, galería, información)
* Uso de imágenes, videos y contenido multimedia
* Interfaz responsive (adaptable a móviles y PC)
* Efectos visuales y animaciones

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (para interacciones visuales)

---

## 🎯 Objetivo

Este proyecto tiene como finalidad:

* Practicar diseño de interfaces (UI)
* Mejorar habilidades en desarrollo frontend
* Simular una tienda online profesional
* Crear un proyecto visual atractivo para portafolio

---

## 📂 Estructura del proyecto

* `index.html` → Página principal
* `style.css` → Estilos generales
* `contacto.html` → Página individual del contacto
* `contacto.css` → Estilos de la página contacto
* `catalogo.js`  → Carga dinámica de información de los juegos
* `img/` → Imágenes y recursos multimedia
* `modal.js` → Ventana emergente
